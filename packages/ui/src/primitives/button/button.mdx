# Button

Displays a button or a component that looks like a button.

## Preview

<Canvas of={ButtonStories.Secondary} />

<Controls of={ButtonStories.Secondary} />

## Usage

export const importCode = `import { Button } from "@buildit/ui/button";`

export const usageCode = `<Button variant="secondary">Button</Button>`

<Source code={importCode} language='ts' dark />
<Source code={usageCode} language='tsx' dark />

## Helper

You can use the `buttonVariants` helper to create a link that looks like a button.

export const linkImportCode = `import { buttonVariants } from "@buildit/ui/button";`

<Source code={linkImportCode} language='ts' dark />

export const linkUsageCode = `<Link className={buttonVariants({ variant: "secondary" })}>Click here</Link>`

<Source code={linkUsageCode} language='tsx' dark />

Alternatively, you can set the `asChild` parameter and nest the link component.

export const asChildCode = `<Button asChild>
  <Link href="/login">Login</Link>
</Button>`

<Source code={asChildCode} language='tsx' dark />
